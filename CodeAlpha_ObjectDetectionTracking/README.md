# Object Detection and Tracking using YOLOv8

## 📌 Project Overview

This project performs real-time object detection and tracking using YOLOv8 and OpenCV. The system captures live video from a webcam, detects multiple objects in real time, and displays tracking boxes with labels and confidence scores.

The project demonstrates how computer vision and deep learning can be used to identify and track objects such as people, mobile phones, laptops, bottles, chairs, and many other everyday objects.

---

## 🚀 Features

* Real-time object detection
* Object tracking across video frames
* Webcam integration using OpenCV
* Bounding box visualization
* Object labels with confidence scores
* Lightweight YOLOv8 model for fast performance

---

## 🛠️ Technologies Used

## ⚙️ Installation

### Install Required Libraries

```bash
pip install ultralytics opencv-python
```

---

## ▶️ How to Run

Run the Python file:

```bash
python object_detection.py
```

The webcam will open and start detecting and tracking objects in real time.

Press **Q** to close the application.
